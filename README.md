## FAQ

**为什么演示网址界面和`demo`文件夹里的不一样**

因为我用vue重写了(https://github.com/5ime/vue-page)

**网址中包含特殊字符导致GET请求无法传递正确的参数值**

传递的参数中包含`#&=`之类的，可能无法正确传递参数值，建议使用`POST请求`或`urlencode编码`后进行GET请求

**关于有些视频平台解析失败**

有些平台需要cookie，请手动更新cookie，如果还是解析失败，请提交issues
